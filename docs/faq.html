Frequently Asked Questions

Does it really run on Windows XP?
Yes. GxPT is a native Windows application targeting .NET Framework 3.5, so it runs on Windows XP and every version of Windows since.
What do I need to run it?
Windows XP or later and the .NET Framework 3.5 runtime. You'll also need an OpenRouter.ai API key to talk to AI models.
Which AI models are supported?
A huge range of frontier and open models through the OpenRouter.ai API — you choose which model to use per conversation.
Is my data private?
Conversations are stored locally on your machine. You can enforce Zero Data Retention (ZDR) on a per-conversation basis so prompts aren't retained upstream.
What is MCP, and what can GxPT do with it?
The Model Context Protocol lets GxPT call external tools: agentic coding (file, git, and shell operations), web search, and GitHub access. The agent chains these tool calls autonomously until your task is done.
Is it safe to let the agent run tools?
Every tool call is gated by an in-app approval prompt, and file, git, and command tools stay confined to the working folder you choose.
How much does it cost?
GxPT itself is free and open source. You pay only for the AI usage billed through your own OpenRouter.ai account.
Where do I report bugs or request features?
On the GitHub repository.
