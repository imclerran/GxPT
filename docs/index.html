Welcome

A native chatbot client and coding agent for Windows XP. GxPT brings agentic workflows to the era of Luna and Aero — autonomously chaining tools for agentic coding and web search via the Model Context Protocol (MCP), with per-conversation privacy controls.

Features

  • Agentic Workflows — autonomously chains tool calls until your task is done: agentic coding (file/git/shell), web search & fetch, and GitHub access via MCP.
  • Skills — teach GxPT reusable workflows that bundle instructions and optional scripts; author your own with the built-in skill-writer and enable them with /skills, /skill, and /use.
  • Persistent Memory — records durable facts about each workspace and recalls them in later conversations, so context carries over.
  • AGENTS.md Support — project instructions in your workspace root's AGENTS.md are injected into the agent's system prompt automatically.
  • Prompt Caching — requests are zoned for provider prompt caches, with sticky routing that keeps follow-ups on the warm cache — up to ~90% off input costs on long agentic sessions.
  • Usage Status Bar — a live context meter plus running cost and cache-savings totals for every conversation.
  • Tool Approval & Sandboxing — every tool call is gated by an in-app approval prompt; file/git/command tools stay confined to the workspace you choose.
  • Slash Commands — quick / actions with autocomplete: /model, /tool, /new, /export, and /compact.
  • Markdown & Syntax Highlighting — rich Markdown rendering plus code highlighting for 50+ languages.
  • Privacy & Local Storage — conversations stored locally; enforce Zero Data Retention (ZDR) per conversation.
  • Frontier Model Support — a huge range of AI models through the OpenRouter.ai API.
  • Legacy Compatibility — runs on Windows XP and .NET 3.5.
